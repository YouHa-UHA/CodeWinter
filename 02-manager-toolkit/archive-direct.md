# Archive Direct 模板

适用于小范围、低风险、边界清晰且最终结果已经确认保留的任务。

```text
本次任务结果已确认保留，请执行归档分类，并且只写入可长期复用的信息。

先分类：
1. stable
2. recent
3. evidence
4. decision
5. handoff
6. discard

再决定写入：
1. stable -> starter、控制面、service manifests、chain cards
2. recent -> service logs
3. evidence -> 40-evidence
4. decision -> 50-decisions
5. handoff -> 30-handoffs
6. discard -> 不归档
```

# Deliverables 正式输出

`05-deliverables` 是面向人的正式输出层。

推荐结构：
1. `governance/YYYY-MM-DD_<topic>/`
2. `tasks/<service-or-project>/YYYY-MM-DD_<task-key>/`

每个主题或任务目录建议固定包含：
1. `index.md`
2. `thread-outputs/`
3. `final/`

规则：
1. 线程草稿写入 `thread-outputs/`。
2. 当前有效正式版本写入 `final/`。
3. deliverables 层不是默认执行知识层。
